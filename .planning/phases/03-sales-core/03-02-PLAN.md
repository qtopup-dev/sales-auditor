## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client → POST /api/sales | productId, mopId, receiver, notes from untrusted client request body |
| Client → PATCH /api/sales/:id | field name and value from untrusted client; could attempt to inject status, organizationId, etc. |
| Client → POST /api/sales/:id/void | saleId from URL param; role check required |
| Client → GET /api/sales/:id/audit | saleId from URL param; admin-only data |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-03 | Tampering | PATCH /api/sales/:id field injection | mitigate | `ALLOWED_PATCH_FIELDS` allowlist enforced via `body('field').isIn([...])` express-validator check — rejects status, organizationId, createdById, etc. |
| T-03-04 | Elevation of Privilege | POST /api/sales/:id/void by moderator | mitigate | `requireRole('admin')` middleware on void route — returns 403 before handler executes |
| T-03-05 | Elevation of Privilege | PATCH /api/sales/:id on another user's row | mitigate | Inside $transaction: `sale.createdById === req.session.userId! && requestingUser.canEdit` check; throws 403 FORBIDDEN if not owner or canEdit=false |
| T-03-06 | Elevation of Privilege | GET /api/sales/:id/audit by moderator | mitigate | `requireRole('admin')` middleware on audit route — returns 403 before handler executes (D-14) |
| T-03-07 | Tampering | Integer overflow on saleId param | mitigate | `param('id').isInt({ min: 1 })` rejects non-integer and negative IDs in patchSaleValidation and voidSaleValidation |
| T-03-08 | Repudiation | Audit log oldValue from request body instead of DB | mitigate | oldValue is read from `tx.sale.findFirst` result BEFORE update, not from `req.body.value` — prevents audit forgery via numeric coercion or whitespace manipulation |
| T-03-09 | Elevation of Privilege | salesRouter mounted outside protectedRouter | mitigate | salesRouter is mounted via `protectedRouter.use('/sales', salesRouter)` — inherits requireAuth from `app.use('/api', requireAuth, protectedRouter)` |
